# Docker Guide

## Simple steps in a docker workflow

- Write the ```Dockerfile```
- Build the IMAGE
- Deploy CONTAINERS from the built IMAGE

### To pull an IMAGE from ```dockerhub```
> docker pull <IMAGE_NAME>

### To run the image:
> docker run <IMAGE_NAME>

### To list

Containers
> docker container ls

Images
> docker image ls
\
or
\
> docker images
